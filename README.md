# Diwali Sales Analysis

## Project Overview
This project performs an **Exploratory Data Analysis (EDA)** on Diwali sales data to improve customer experience and increase revenue.

## Tools Used
* **Python** (Pandas, NumPy)
* **Matplotlib / Seaborn** (for data visualization)
* **Jupyter Notebook**

## Key Insights
- Identified the most popular product categories.
- Analyzed purchasing power based on gender and age groups.
- Visualized state-wise sales distribution.

## How to Run
1. Open the `Diwali_Sales_Analysis.ipynb` file in Jupyter Notebook or VS Code.
2. Ensure `Diwali Sales Data.csv` is in the same directory.
3. Run all cells to see the visualizations.
